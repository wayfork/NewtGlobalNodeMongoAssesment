Please do the following steps to run the application once code is downloaded

step1 : NodeJS & mongoDB should be installed on the machine
step2 : create assesmentDB in mongodb
step3 : go inside the downloaded application directory
step4 : open command prompt or bash on the path
step5 : run "npm install"
step6 : run "node app"
step7 : open browser
step8 : search for http://localhost:3000
step9 : you are all done, lets play with the data
