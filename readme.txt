Please do the following steps to run the application once code is downloaded

step1 : NodeJS & mongoDB should be installed on the machine
step2 : go inside the application directory
step3 : open command prompt or bash on the path
step4 : run "npm install"
step5 : run "node app"
step6 : open browser
step7 : search for http://localhost:3000
step8 : you are all done, lets play with the data
